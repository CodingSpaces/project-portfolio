Project Portfolio allows you to create pages for projects of interest.  These can either be your project portfolio or outside projects of interest.

== Description ==

Project Portfolio allows to create a unique page for displaying project information to include:
  - description 
  - project abstract 
  - project URL
  - source code url
  - organization
  - logo or screenshot

== Installation ==

1. Upload `project-portfolio` folder to the `/wp-content/plugins/` directory
2. Activate the plugin through the 'Plugins' menu in WordPress
